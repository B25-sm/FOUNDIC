Foundic Favicon Test

If you can see the Foundic "F" logo in your browser tab, the favicon is working correctly!

The favicon should show a golden circle with a dark "F" letter inside.

If you still see the React icon:

  • Hard refresh the page (Ctrl+F5 or Cmd+Shift+R)
  • Clear browser cache
  • Close and reopen the browser tab
  • Try opening in an incognito/private window
